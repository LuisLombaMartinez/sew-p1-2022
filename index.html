¿Qué es Laravel?

Introducción

Laravel es un framework de código abierto para desarrollar aplicaciones y servicios web con PHP 5, PHP 7 y PHP 8. Su filosofía es desarrollar código PHP de forma elegante y simple, evitando el "código espagueti". Fue creado en 2011 y tiene una gran influencia de frameworks como Ruby on Rails, Sinatra y ASP.NET MVC.

Historia

Taylor Otwell creó Laravel en un intento por proveer de una alternativa más avanzada a CodeIgniter framework, el cuál no proveía de algunas características como un soporte nativo para la autenticación y autorización. La primera versión de Laravel, incluía soporte nativo para autenticación, localización, modelos, vistas, sesiones y enrutamiento, pero no tenía soporte para controladores, lo que no lo hacía un verdadero framework MVC.

Las siguientes versiones, fueron corrigiendo estos detalles y a día de hoy Laravel funciona con la idea de utilizar un patrón MVC, estableciéndose como un estándar, incluso en otros lenguajes de programación.

En la actualidad es un framework establecido con gran cantidad de usuarios.
